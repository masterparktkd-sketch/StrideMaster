# StrideMaster: AI Running Program Builder

StrideMaster is a professional-grade tool designed to generate comprehensive, personalized AI prompts for running training, nutrition, and recovery.

## 🚀 Features

- **Professional Architecture**: Built on elite coaching principles (80/20 rule, 10% progression).
- **Intelligent Modules**: Custom prompts for Training, Nutrition, Injury Prevention, Recovery, and Pacing.
- **Bilingual Support**: Full support for English and Korean.
- **Modern UI**: Clean, responsive interface built with React, Tailwind CSS, and Framer Motion.

## 🛠️ Tech Stack

- **Frontend**: React 19, TypeScript, Vite
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion (motion/react)
- **Icons**: Lucide React

## 📦 Deployment to Vercel

1. **Push to GitHub**: Export your project to a GitHub repository.
2. **Connect to Vercel**: Import the repository into Vercel.
3. **Build Settings**:
   - Framework Preset: `Vite`
   - Build Command: `npm run build`
   - Output Directory: `dist`
4. **Environment Variables**: If you add Gemini API integration later, add `GEMINI_API_KEY` in the Vercel dashboard.
